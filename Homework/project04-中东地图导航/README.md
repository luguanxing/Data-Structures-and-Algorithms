## 项目简介
中山大学东校区图导航软件

## 项目要求
1.从中山大学东校区的平面图中选取10个景点，抽象成一个无向带权图<br>
2.为来访客人提供图中任意景点的问路查询，即查询任意两个景点之间的一条最短的简单路径<br>
3.区分汽车线路与步行线路<br>
